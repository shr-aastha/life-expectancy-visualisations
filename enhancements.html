Enhancements

  1. Enhancement 1: HTML details and summary elements

    How it goes how it goes beyond the basic requirements of the assignment?

    The HTML summary and details elements were not taught in this unit; it was learnt through self-research.

    What code is needed to implement the feature?

    It can be implemented by enclosing the caption for the hidden content within the summary element and enclosing the content to be hidden in the details element. The summary tag must be enclosed within the details element. The code to implement this feature is provided below: 

    
            <details>
                <summary>
                    <h3>Caption</h3>
                </summary>
                <p>Content to be hidden</p>
            </details>

    References to any third party sources for the technique, (e.g. URL)

    The MDN Web Docs references for the Details disclosure element and the disclosure summary element were referenced to implement this enhancement.

    Hyperlink to where the enhancement has been implemented in the website

    The details and summary elements have been used in the Jobs page to toggle the visibility of information relating to key responsibilities, required and preferred skills.

  2. Enhancement 2: Implementing tabs with the use of the element + element CSS Selectors

    How it goes how it goes beyond the basic requirements of the assignment?

    The use of CSS selectors were briefly discussed and implemented during the tutorials, however, the use of adjacent sibling combinator for creating tabs were not taught in this unit.

    What code is needed to implement the feature?

    It can be implemented by controlling the visibility of each content through the use of radio buttons, adjacent sibling combinators and the display property. Based on the radio button that is clicked, the content located below the radio button in the HTML file is shown. The CSS code to implement this feature is provided below:

    .tabradio { display: none; }
    .tabcontent { order: 1; width: 100%; border-bottom: 3px solid #dddddd; line-height: 1.5; display: none; font-size: 0.9em; }
    .tabradio:checked + .tablabel + .tabcontent { display: initial; }

    References to any third party sources for the technique, (e.g. URL)

    The code snippets on CodePen were referenced to implement this enhancement.

    Hyperlink to where the enhancement has been implemented in the website

    The tabs have been implemented in the About page to present my personal interests and professional experience.
